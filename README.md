# Timetable App

This is a simple educational timetable app.

**Free for personal or educational use only. Do NOT sell or distribute for profit.**

## How to Run
1. Download `timetable.exe` from Releases (or run the Python source code).  
2. Double-click `.exe` to open the app.  
3. Follow the on-screen instructions to add lessons and manage your timetable.
